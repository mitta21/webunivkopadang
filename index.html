We Are Good In

We use the assistance of applications and references from the following sources

QGIS

QGIS adalah perangkat lunak sumber terbuka untuk analisis dan visualisasi data geografis, mendukung berbagai format dan plugin untuk memperluas fungsionalitas.

Indonesia Geospatial Portal

website yang kami gunakan untuk mengambil data dari wilayah kota Padang. Dengan ini kami bisa mendapatkan database geografis yang ingin kami teliti.

Visual Studio Code

Dalam pembuatan website ini, aplikasi VSCode digunakan melakukan pemrograman, dengan menggunakan HTML, CSS, dan javascript.

GitHub

Dalam menggabungkan web qgis dengan index.html serta mempublikasikan menjadi sebuah web yang dapat diakses atau dikolaborasikan oleh banyak orang.

Google Maps

Kami menggunakan Google Maps untuk membuat titik yang tepat serta menambahkan data yang kurang seperti latitude dan longitude, juga nama jalan serta menyesuaikan titik supaya lebih akurat pada peta.

YouTube

Menggunakan referensi dari YouTube untuk membantu dalam pembuatan peta.

Universitas

Kota Padang merupakan pusat pendidikan di Sumatera Barat dengan banyak perguruan tinggi terkemuka. seperti Universitas Andalas, UNP, Politeknik ATI Padang, Universitas Muhammadiyah Sumbar, UPI Lubukbegalung, Sekolah Kepolisian Negara, dll.

Tampilan

Pada web GIS yang telah dirancang, kita bisa melihat dan menyesuaikan peta sesuai kebutuhan. dengan mengaktifkan atau tidak layers yang disediakan, serta dapat langsung mencari di kolom pencarian.

Titik Lokasi

Setiap titik lokasi yang disediakan dan di klik akan langsung memberi informasi berupa gambar, juga bisa memberi akses ke web lain apabila ingin mengetahui universitas yang dicari lebih dalam lagi.
